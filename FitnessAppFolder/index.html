FITNESS APP
FOR GYMS, TRAINERS, CLIENTS

FEATURES

6x6 is workout planner for those seeking strength gains

Challenge

The application was developed for a hackathon during my stay in Lisbon. My Mobile Programming Professor tasked us to building an application to help combat the lasting effects of COVID.

Problem

After recovering from Covid, I realised I was weaker than before so I had to do something.

Solution

I implemented my workout routine that helped me get back in shape.

Technologies

Built for all Android devices, using Kotlin

Google Firebase

Will use Firebase authentication and real time databse

Good Foundation

Concept created using Adobe XD and following material design

Easy to Use

The UI was built around simplicity, to let you focus on your exercises and not be an inconveniences
